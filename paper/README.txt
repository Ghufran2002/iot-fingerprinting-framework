HOW TO COMPILE ON OVERLEAF
================================================================

STEP 1 — Upload
  • Go to https://overleaf.com → New Project → Upload Project
  • Zip this entire folder and upload the ZIP
    (or upload main.tex + references.bib + figures/ separately)

STEP 2 — Set Compiler
  • In Overleaf: Menu → Compiler → select "pdfLaTeX"
  • LaTeX version: TeX Live 2023 (default)

STEP 3 — Compile
  • Click "Recompile" — the paper will build automatically
  • If you see BibTeX warnings on first compile: recompile once more
    (pdflatex → bibtex → pdflatex → pdflatex is the standard sequence;
     Overleaf handles this automatically after the first pass)
